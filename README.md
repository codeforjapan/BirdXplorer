# BirdXplorer

## Description

BirdXplorer is software that helps users explore community notes data on X
(formerly known as Twitter).

## Development

### Requirements

- [Python](https://www.python.org/) (v3.10.12)
- [PostgreSQL](https://www.postgresql.org/) (v15.4)

### Installation

```bash
pip install -e ".[dev]"
```

### Testing

```bash
tox
```
